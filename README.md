# RadioDJListenerTool
Kleines Windows-Tool für Laut.fm zur automatischen Abfrage der aktuellen Hörerzahl.
🎧 RadioDJ Listener Tool für Laut.fm

Dieses kleine Windows-Tool ruft automatisch die aktuelle Hörerzahl eines Laut.fm Senders ab und schreibt diese in eine Textdatei.

Ideal zur Weiterverwendung in RadioDJ, Overlays oder anderen Automation-Setups.

## Funktionen
- automatische Abfrage der aktuellen Hörerzahl
- frei einstellbares Aktualisierungsintervall
- Ausgabe in eine Textdatei
- einfache Konfiguration
- Einstellungen werden gespeichert
- Minimieren in den System-Tray
- Installer mit Ersteinrichtung

## Verwendung
1. Sendernamen eingeben
2. Ausgabedatei festlegen
3. Start klicken

## Hinweis
Beim ersten Start kann Windows eine Sicherheitswarnung anzeigen, da das Tool noch keine digitale Signatur besitzt.

Das ist bei neuen Anwendungen normal.
Das Programm kann über „Weitere Informationen“ → „Trotzdem ausführen“ gestartet werden.
